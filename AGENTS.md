# Project Instructions

## Testing

- `npm run type-check` — TypeScript check
- `npm run test` — Jest unit tests
- `npm run smoke` — smoke subset
- `npm run verify` — type-check + test + smoke (full local pre-commit)
- `npx playwright test` — e2e suite

## Build & Run

```bash
npm run dev
```

## Dependencies

```bash
npm install
```
