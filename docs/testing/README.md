## Key Testing Scenarios

### Scenario 1: Happy Path (30 minutes)
Test a complete conversation from start to finish with early exit.

**Flow:** INITIAL → LENS_SELECTION (A) → QUESTIONING (3-5 questions) → Early Exit (B) → EXTRACTION

**Tests:** Core flow, confidence assessment, early exit, extraction display

---

### Scenario 2: Maximum Questions (15 minutes)
Test the 10-question limit with auto-extraction.

**Flow:** INITIAL → LENS_SELECTION → QUESTIONING (10 questions) → Auto-EXTRACTION

**Tests:** Question limit enforcement, auto-transition

---

### Scenario 3: Exploration Loop (20 minutes)
Test exploration resume functionality.

**Flow:** INITIAL → LENS_SELECTION → QUESTIONING → EXTRACTION → "Explore further" → QUESTIONING → EXTRACTION

**Tests:** Phase transitions, resume questioning, updated extraction

---

### Scenario 4: Different Lenses (45 minutes)
Test multiple lenses to verify different framing.

**Test:** Run Scenario 1 with Lens A, then Lens C, then Lens E

**Verify:** Questions feel different, framing is lens-appropriate
